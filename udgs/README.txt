udgmaker - a basic program which creates and saves user defined graphics

Copyright (C) 2002 Malcolm Tyrrell

Instructions:
=============

1. Making "udgmaker.tap": type "make" at the command-line.
2. Run an emulator, and load the program in "udgmaker.tap".
3. Create a new tape file in the current directory called "MazezaM2.tap" for
   writing to. How this is done depends on your emulator.
4. Run the program and press a key.
5. When the program has finished, stop the tape and close the emulator.
